Now.ts UI is a component library built on top of [shadcn/ui](https://ui.shadcn.com/) to help you build fullstack React applications faster. It provides pre-built components for authentication, forms, dialogs, and more.

Components are available via the standard shadcn CLI:

```bash
npx shadcn@latest add https://ui.nowts.app/r/<component>.json
```

For example, to install the [Submit Button](/docs/components/submit-button) component, you can run:

```bash
npx shadcn@latest add https://ui.nowts.app/r/submit-button.json
```
